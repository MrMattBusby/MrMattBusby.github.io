## How-to

1. Name repo <username>.github.io
1. Enable pages
1. Use default template for Jekyll
   1. Default _config.yml
   1. Default _layouts/default.html, ChatGPT helps
   1. Default assets/css/*
      1. Customize the styles.css with whatever you like, find, or make with LLM
1. Create resume as index.md, gets auto-converted to index.html
   1. Use a some standard yml header
   1. Use a few IDs like {: .id } *flatlist* or *twocollist* after lists for custom formatting
1. Updates static website on deploy part of push build
1. Could optionally make manual workflow to print html to pdf
